## 🔍 Financial Fraud Detection & Monitoring System

## 📌 Project Overview

This project presents an end-to-end Financial Fraud Detection and Monitoring System built using SQL and Power BI.

The objective is to simulate a real-world fraud detection environment by designing a relational database, generating transactional data, detecting fraudulent patterns, and building a monitoring dashboard for decision-making.

Unlike basic analysis projects, this system was built from scratch — including database design, data generation, SQL logic, and visualization — to demonstrate practical fraud analytics and risk monitoring capabilities.
