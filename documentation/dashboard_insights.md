# 📊 Dashboard Insights

## 📌 Dashboard Summary

An interactive Power BI dashboard was developed to monitor fraud activity.

---

## 📊 Key Metrics
- Total Fraud Exposure: 17.36M  
- Fraud Rate: 47.6%  
- Number of Fraud Cases  
- High-Risk Accounts  

---

## 🔍 Key Insights
- Fraud is concentrated within a small number of accounts  
- High-value transactions are more likely to be fraudulent  
- Specific locations (e.g., Lagos) show higher fraud activity  
- Rapid and repeated transactions indicate suspicious behavior  

---

## 📌 Business Impact
- Enables quick identification of fraud hotspots  
- Supports faster decision-making  
- Improves visibility into financial risk
